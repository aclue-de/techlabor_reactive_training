# reactive-spring-webflux
Spring Webflux


##Start Mongo DB
Pull image for MongoDB

Prerequisite: Docker

```docker pull mongo:latest```

Start Container

```docker run -d --name mongo-on-docker -p 27017:27017 mongo```
